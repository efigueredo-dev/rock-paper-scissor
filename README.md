# rock-paper-scissor

This is a console game where the user plays 5 round of the game "Rock, Paper, Scissor" against the computer.

The rules are very simple:

- Rock beat Scissor
- Scissor beat Paper
- Paper beat Rock

## Important

Before playing this game, you'll need to open the Developer's Tool and go to the console's section. Since this game was developed using the Mozilla Firefox browser, you can open the console pressing ***Ctrl + Shift + K***.

If not, follow the next steps:

1. Open the browser's menu
2. Go to *More tools* -> *Developer's Tools* (or press Ctrl + Shift + I)
3. Go to the *Console* section
4. Now you can play the game. I wish you good luck!
